# NHRSJWF Website

**National Human Rights & Anti Corruption Social Justice Welfare Foundation**

A modern, multi-page NGO website built with React, TypeScript, Tailwind CSS, Vite, and React Router.

## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🛠️ Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Poppins, Montserrat, Open Sans
